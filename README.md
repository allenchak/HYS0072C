# HYS0072C
HYS0072C NF62L-VC 1.5 inch LCD display using HT1621

# Pinout
| Pin | Name | Description | Arduino |
| --- | --- | --- | --- |
| 1 | VCC | Vcc | 5V |
| 2 | GND | GND | GND |
| 3 | Data | Data | D13 |
| 4 | WR | WR | D12 |
| 5 | CS | Chip select | D11 |
| 6 | BL | Backlight LED, HIGH for ON, LOW for OFF; Floating for always on | D10 |
